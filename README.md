# LeetCode Solutions 🚀

A collection of LeetCode problems solved in **Java**. This repository is automatically updated using the **LeetSync** extension.

## 🛠️ How It Works
* **Automatic Pushes:** Every successful code submission is automatically pushed to this repository.
* **Organization:** Each problem gets its own folder containing the optimal Java solution file.

## 🔧 Sync Troubleshooting (If code doesn't update)
If your LeetCode streak updates but the code does not appear here, verify these three quick settings:

1. **URL Format:** Check your LeetSync popup. The repository link must look exactly like this:  
   `https://github.com`  
   *(Make sure there is NO trailing slash `/` or `.git` at the end)*
2. **Repository Visibility:** If this repository is **Private**, ensure you granted LeetSync full write permissions during the GitHub login step.
3. **Trigger Push:** LeetSync only pushes code when you click **Submit** and get the green **"Accepted"** screen. It will not push on "Run Code".
